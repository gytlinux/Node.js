# Nodejs module - sleep

休眠，同shell中sleep

### Install

```
npm install sleep --save
```

### Usage

```
var sleep = require('sleep');
sleep.sleep(n)  //sleep for n seconds
sleep.msleep(n) //sleep for n miliseconds
sleep.usleep(n) //sleep for n microseconds (1 second is 1000000 microseconds)
```
